# 训练管理系统架构设计

## 系统架构图

```markdown
+-------------------+        +-----------------+        +-------------------+
|                   |        |                 |        |                   |
|   客户端(浏览器)   |<------>|   Web服务器     |<------>|   数据库服务器     |
|                   |   HTTP |  (SpringBoot)   |   JDBC |    (MySQL)        |
+-------------------+        +-----------------+        +-------------------+
        |                            ^
        |                            |
        v                            v
+-------------------+        +-----------------+
|                   |        |                 |
|   静态资源服务     |        |   缓存服务      |
|   (Vue.js)        |        |   (内存缓存)    |
+-------------------+        +-----------------+


+-----------------------------------------------------------+
|                       后端架构分层                         |
+-----------------------------------------------------------+
|   控制层 (Controller)  |  处理HTTP请求，返回HTTP响应       |
+------------------------+-------------------------------------+
|   服务层 (Service)     |  实现业务逻辑                      |
+------------------------+-------------------------------------+
|   数据访问层 (Mapper)  |  数据库访问接口                    |
+------------------------+-------------------------------------+
|   模型层 (Model)       |  数据实体                          |
+-----------------------------------------------------------+


+-----------------------------------------------------------+
|                       前端架构分层                         |
+-----------------------------------------------------------+
|   视图层 (View)        |  用户界面展示                     |
+------------------------+-------------------------------------+
|   路由层 (Router)      |  页面路由控制                      |
+------------------------+-------------------------------------+
|   状态管理 (Vuex)      |  应用状态管理                      |
+------------------------+-------------------------------------+
|   API服务 (Axios)      |  与后端通信                        |
+-----------------------------------------------------------+
```
